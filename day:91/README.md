## 266. Palindrome Permutation
[Leetcode link](https://leetcode.com/problems/palindrome-permutation/)

Given a string s, return true if a permutation of the string could form a palindrome.

 

### Example 1:

Input: s = "code"
Output: false

### Example 2:

Input: s = "aab"
Output: true

### Example 3:

Input: s = "carerac"
Output: true
